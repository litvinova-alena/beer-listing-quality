# beer-listing-quality

A learning project focused on analyzing and evaluating the quality of beer product listings.

The goal of this project is to explore data quality techniques, identify issues in beer listings across 3 different cities (Prague, Munich, and Dublin), calculate quality metrics, and build an automated quality scoring framework.

---

## Objectives

- Assess the quality of beer listings
- Detect missing, invalid, and inconsistent data
- Calculate an overall Listing Quality Score
- Analyze data using SQL and Python
- Visualize quality metrics
